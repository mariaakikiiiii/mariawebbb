What is a CSS Rule?

A CSS rule defines how HTML elements should be displayed on a webpage. Each rule consists of a selector and a set of declarations. The selector determines which elements are affected, while the declarations specify the style changes.

Basic Structure

selector { property: value; }

For example: 

              p {
                color: blue;
                font-size: 16px;
              }

This CSS rule makes all <p> elements appear with blue text and a font size of 16 pixels (Mozilla, 2023).

Types of Selectors

  1. Element selectors - apply styles to a specific HTML element (e.g., p, h1).
  2. Class selectors - apply styles to elements with a specific class attribute (e.g., .example).
  3. ID selectors - apply styles to an element with a unique ID (e.g., #header).
  4. Universal selectors - apply styles to all elements (*).

Importance of CSS Rules

  • They separate content (HTML) from presentation (CSS).
  • They allow for consistent design across multiple pages.
  • They improve maintainability and accessibility (W3C, 2023).
